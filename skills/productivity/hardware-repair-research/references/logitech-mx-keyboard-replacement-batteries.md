# Logitech MX Keyboard Replacement Battery Notes

Session context: user clarified that an MX keyboard worked for years and then suddenly could not keep a charge, then asked for Amazon replacement battery options.

## Diagnostic interpretation

When an MX keyboard has behaved normally for years and then suddenly drains quickly or reports low battery soon after charging, treat **battery aging / Li-Po end-of-life** as more likely than normal backlight drain.

Useful checks before recommending parts:

1. Fully charge with a known-good USB-C cable/charger.
2. Disable backlighting for a full cycle or at least 24–48 hours.
3. Optionally turn the keyboard off overnight after charging.
4. If it still drains quickly, especially while off or with backlighting disabled, suspect the internal cell.
5. Warn the user to stop charging/using it if there is swelling, heat, odor, or case separation.

## Compatibility details to verify

Ask for or inspect the underside label before buying. MX-family names are similar but use different packs.

- **MX Keys / MX Keys S full-size**
  - Common keyboard model: `YR0073`
  - Common battery part: `533-000177`
  - Typical pack: `3.7V 1500mAh / 5.55Wh`
  - Typical size reported by Amazon listings: about `85.1 × 19.2 × 7.2 mm`

- **MX Keys Mini**
  - Common keyboard model: `YR0084`
  - Common battery part: `533-000200`
  - Typical pack: `3.7V 1050mAh`

- **MX Mechanical / MX Mechanical Mini**
  - Some listings claim compatibility with `AHB355085PCT` / `3000mAh` packs, often shared with Logitech G915/G913/G715 family. Verify exact model and connector carefully; do not assume full-size MX Keys packs fit.
